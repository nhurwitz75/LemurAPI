#Project Title - Lemur API 
API for all the Lemur enthusiasts 

##Description 
This API holds key data on different Lemur species 

##Usage 
Click on this link to access the API
https://lemurapi.onrender.com/docs

##Features 
Lemur species common name, mood based on color palette, and conservation status
